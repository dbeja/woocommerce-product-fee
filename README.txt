== Description ==
This plugin increases WooCommerce cart total by X% if a certain product is in the cart.

The percentage fee and cart label can be customised on WooCommerce settings.

== Installation ==
To install the plugin:

1. Upload `cproductfee` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
